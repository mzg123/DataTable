操作演示--提现

步骤1：提现入口（3种路径）

1） 实名认证后点击“我的账户”进入“账户总览”，点击“提现”进入充值页：

2） 鼠标移入我的账户，选择下拉菜单中“提现”，进入充值页：

3） 进入“账户总览”，鼠标移入“资金管理”，点击“提现”，进入充值页：

步骤2：提现

进入提现页，输入提现金额，点击“提现”进入“提现成功”页:

步骤3：提现成功

提现完成后，跳转提现成功，需要1-2个工作日内到账，点击“查看记录”跳转“交易记录”；
