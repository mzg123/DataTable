资费说明

出借服务

1、银谷在线财富管理计划有几个系列，收益率为多少？

银谷在线财富管理计划分为两个系列，一个为东方红财富系列，主要有东方红1号、东方红2号、东方红3号，期限为12 -24个月，预期年化收益率为12%-？%，一类为任逍遥系列，主要有任逍遥1号、任逍遥2号、任逍遥3号，期限1-6个月不等，年化收益率7%-9%；满足出借用户短、中、长不同需求。

2、出借成功后是否可以撤销？

1) 出借债权列表中债权，不可以撤销，在限定时间内出借债权的募集进度没有达到100%，这部分资金会返还在账户余额中。

2) 出借转让中债权，未转让部分可以撤销，转让成功部分不可以撤销。操作步骤：我的账户-债权管理-回款中债权。

3、冻结金额是怎么回事？

冻结金额分为募集中冻结和满标待审冻结、提现冻结、其他冻结。

1) 募集中冻结：指在您出借的债权募集期内，募集进度尚未达到100%，您的出借金额会被冻结，若在募集期结束时债权募集进度没有达到100%，即流标，这部分资金将返还至您的账户余额。

2) 满标待审冻结：在您的出借债权募集进度达到100%后，系统会对债权进行审核，在审核期间，这部分资金将处于冻结的状态，待审核通过后，这部分冻结金额将出借给借款人，可以在我的账户-回款中债权查看。

3) 提现冻结：指您提交提现申请后，这部分资金会被冻结，待提现审核通过后，这部分资金将由第三方支付公司划拨至您绑定的银行卡，提现时效为T+2日。

4) 其他冻结：除了以上冻结的状态之外的其他类型，包括但不限于转让服务费，催收服务费，提现手续费等。
