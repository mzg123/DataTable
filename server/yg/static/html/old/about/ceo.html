银谷CEO

  • 孙敏 银谷创始人

    银谷创始人兼总裁，天使投资人。自上世纪九十年代末开始创业，曾先后涉足航空物流、矿产、房地产及金融行业。

    有感于金融市场的新形势，并凭借对其的敏锐观察，于2007年创立银谷公司。广泛开展财富管理、信用风险评估与管理、信用数据整合等相关服务。致力于搭建中国信用体系，释放信用价值，为促进互联网金融市场繁荣与健康做出贡献！

    银谷公司自创立以来，已在全国开设了百余家直营分公司，其公司规模、运营效率、客户口碑、社会责任等均在社会占有一定地位，并得到社会广泛认可，帮助了更多大众实现财富的管理与生活品质的提升！

  • 李峰屹 银谷创始人

    银谷创始人，资深互联网金融及网络营销专家，天使投资人。自上世纪九十年代末开始创业，凭借深厚的IT行业经验，成功实现了将互联网技术与传统企业相结合的新型发展模式。其独立运营开发的多个平台应用于携程旅行网、去哪儿网、华美地产、三峡燃气等多家上市企业，备受业界关注。先后涉足航空物流、矿产经营、移动互联网，并于2007年引进国外先进的个人信用价值理念和互联网技术，参与创立银谷。基于其强烈的企业社会责任感，以及优秀领导力，李峰屹先生得到社会人士广泛认可。

  • 黄振华 银谷创始人

    金融行业高级法律顾问，精通互联网金融领域相关法律知识。毕业于西北政法大学，北京大学EMBA。执业超过十年，精通民商、经济法律、熟悉大型企业的运作模式、经营管理以及法律实务工作，先后在秦丰农业、合生创展、中新集团、双良集团、国华置业等多家上市公司、大型企业集团担任法律顾问，先后参与、主持了多个500强企业转型、重组、并购、上市、国际合作等领域的法律工作。
