公司介绍

“银谷在线”是一家互联网金融综合服务平台，由东方银谷（北京）投资管理有限公司（简称“银谷”）于2016年推出。银谷创建于2007年，总部设立在繁华的北京CBD商圈，旨在为用户提供集财富管理、信用风险评估与管理、小微借款咨询服务与交易促成平台服务等稳健的财富管理与信用管理服务。

银谷公司拥有银谷财富、银谷普惠、银谷普诚、云钱袋等支柱业务，如银谷财富——财富管理信息咨询服务平台；银谷普惠——中国小微借款咨询服务专业平台；银谷普诚——中国信用风险评估与管理专业服务平台；云钱袋——中国网络借贷信息咨询服务平台。

自2007年成立至今，已在京、沪、津、渝、冀、辽、吉、黑、陕、鲁、闽等地开设了600余家直营分公司，员工数量23000多人，为社会提供了千余就业岗位，为数万家庭及企业提供过全方位、个性化的财富管理信息咨询服务。历经九年多的摸索和历练，银谷积累了大量的媒体及渠道资源，并通过与相关机构的战略合作，为公司在业界赢得了良好的信誉和口碑。

放眼未来，银谷将继续秉承“共享共赢，同信同力，创新创展，立信立人”的企业核心理念，以实际行动践行企业对社会责任的承诺，通过稳健的经营模式与真诚专业的客户服务，充分发挥财富管理的纽带作用，为广大客户创造更高的生活品质与更大的财富享受，为促进中国市场经济的繁荣稳定做出贡献！

融信誉而立天下,创诚信而惠东方！
